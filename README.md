# 💖 Birthday Surprise Website

A romantic, interactive birthday website made with ❤️ for my girlfriend.

This project is a cinematic digital love letter that starts with a secure passcode screen and unlocks into a beautiful surprise experience filled with memories, music, and emotions.

## 🌐 Live Website

👉 https://yourusername.github.io/birthday-website/

## ✨ Features

🔐 Passcode Lock Screen

* Enter a secret 4-digit code to unlock the surprise

🎉 Celebration Animation

* Surprise effect after correct passcode

🎵 Background Music

* Romantic music with play/pause control

💖 Floating Hearts Animation

* Continuous aesthetic love animation

💌 Love Letter (Typing Effect)

* A heartfelt message that types itself

📸 Photo Gallery

* Memories displayed with hover animations

💞 Reasons I Love You

* Cute and meaningful points

🎁 Surprise Button

* Final hidden message

📱 Fully Responsive

* Works on mobile, tablet, and desktop
